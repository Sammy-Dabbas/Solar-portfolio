# 3D Solar System Portfolio

A beautiful interactive 3D portfolio website built with Next.js, Three.js, and React Three Fiber. The portfolio features a solar system theme where each planet represents a different project or section of the portfolio.

Link to view: https://kzmnn7x3spp4poj2tmcj.lite.vusercontent.net/
## Features

- **Interactive 3D Solar System**: Navigate through a fully interactive 3D solar system
- **Dual View Options**: Switch between traditional 2D portfolio view and immersive 3D experience
- **Project Showcases**: Each planet represents a different project with detailed information
- **Responsive Design**: Works on both desktop and mobile devices
- **Animated Elements**: Shooting stars, rotating planets, and dynamic skill constellations
- **Smooth Camera Transitions**: Elegant transitions when focusing on different projects

## Technologies Used

- **Next.js**: React framework for building the application
- **Three.js**: JavaScript 3D library
- **React Three Fiber**: React renderer for Three.js
- **TypeScript**: For type safety and better developer experience
- **Tailwind CSS**: For styling the 2D components
- **Framer Motion**: For animations in the 2D view
- **@react-three/drei**: Useful helpers for React Three Fiber
